INSTALLATION GUIDE

Step 1


Check the compatibility table to see if the Skylands pack is available for your Minecraft version.

(optional) It is recommended to stay updated with the latest stable version, marked in green. Older versions are not supported or maintained.

honorable mention:
Some skylands versions might be both backwards and forward compatible.
Minecraft version Skylands version
1.21.1 3.0
Step 2


Based on your Minecraft version and platform, install a compatible terra version.
Follow >>>this guide on Terra docs.
(paper -> bukkit)
(folia -> bukkit)
Step 3


Please buy the Skylands pack. Downloading should start right after purchase.
>>>Buy skylands pack
Step 4


Place Skylands.zip into packs directory. Path to packs directory will differ based on your platform:
Bukkit:
/plugins/Terra/packs

Fabric:
/config/Terra/packs
Step 5


Create world using Skylands:

>>>Bukkit guide

>>>Fabric guide
skylands id:
SKYLANDS

prefilled bukkit method:


prefilled Multiverse Core command:
Step 6 (optional)


Vanilla structures are not supported, so it is recommended to disable vanilla structure generation.
Step 7 (optional)


Delete your existing old world. This prevents ugly terrain transitions between old and newly generated terrain.

Updating

Unfortunately theres no way to update existing world.

You should reset (delete) your old world when you want to update to newer or degrade to older skylands version.
